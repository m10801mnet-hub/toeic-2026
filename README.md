# TOEIC 500→730 弱点単語アプリ v2

- 収録単語数: 565語
- ユーザーが口頭で出した弱点語を可能な限り収録
- TOEIC730向け追加語を100語以上追加
- 15日ロードマップ
- 最初の10日間: 新規語を覚える
- 最後の5日間: 間違い語と★重要語を復習
- 正解後、画面下に「次の問題へ」ボタンが出ます
- GitHub Pages対応

## 更新方法
このZIPの中身をGitHubのリポジトリ直下に上書きアップロードしてください。


## v3変更点
- 画面下の「次の問題へ」ボタンを廃止しました。
- 正解・不正解の後、緑色になった正解選択肢をもう一度タップすると次の問題へ進みます。
- スクロール操作をほぼ不要にしました。


## v4変更点
- 例文をTOEIC文脈の自然な例文に差し替えました。
- 正解・不正解の後、画面のどこをタップしても次の問題へ進みます。
